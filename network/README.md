# 🎯 Objetivo

Desenvolver domínio prático e conceitual em:

* Implementação de rede interna em **IPv4 e IPv6**
* Segmentação de rede

  * Criação de **VLANs**
  * Criação de **DMZ**
* Configuração de **Firewall** e controle de acesso
* Simulação de **ambientes reais**
* Implementação de **monitoramento**
* Testes de **intrusão simulados**
